# 🥗 VeggieFit_System

A **Personalized Nutrition & Meal Planning System** built using PHP and MySQL to help users generate healthy diet plans based on their goals, preferences, and lifestyle.

---

## 🚀 Project Overview

VeggieFit_System is a web-based application that allows users to:
- Register and login securely
- Generate personalized meal plans
- Track daily nutrition intake
- Monitor health progress
- Access a food database with nutritional information
- Manage user profile and settings

It also includes an **Admin Panel** to manage users, foods, and system operations.

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server Environment:** XAMPP (Apache, MySQL)
- **Email Service:** PHPMailer

---

## ✨ Features

### 👤 User Features
- User registration and login system
- Secure password reset via email
- Personalized meal plan generation
- Daily nutrition tracking
- Health progress monitoring
- Profile management

### 🛠️ Admin Features
- Admin dashboard
- Manage users
- Add, edit, delete food items
- Monitor user progress
- Manage meal plans
